## Installation

First, Check on node version, choose node.js version > 20.9

```bash
nvs
```

Install Project
```bash
npm install
# or
npm i
```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
